# Todo-app
Spring Boot - Vue.js Todo Application

Client klasörü altında npm paket yöneticisi ile gerekli kütüphaneleri indirme işlemini tamamladıktan sonra
npm run serve komutu ile front-end projesi çalıştırılabilir.

Back-end için gerekli kütüphaneleri maven paket yöneticisi kullanarak indirdim.

Server Side: http://localhost:8080
Client Side: http://localhost:8081 

Portlarında çalıştırılmalıdırlar.
